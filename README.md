# 100 Days of Code – Python Bootcamp Projects 

This repository contains all the projects and exercises I built while completing the  
“100 Days of Code: The Complete Python Pro Bootcamp” by 'Dr. Angela Yu' on Udemy.

The goal of this repository is to track my progress, practice core programming concepts,  
and build a strong foundation in Python through hands-on projects.

# Course Overview

- Instructor: **Dr. Angela Yu**
- Platform: **Udemy**
- Duration: **100 Days**
- Focus: Python fundamentals, problem-solving, and real-world projects

# Technologies & Concepts Covered

- Python (Core & Advanced)
- Data Types & Control Flow
- Functions & OOP
- File Handling
- APIs & JSON
- Web Scraping
- Automation Scripts
- Error Handling & Debugging
- Basic Web Development
- Version Control with Git & GitHub
